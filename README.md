# 💔 I'm Sorry Baby

An interactive apology letter generator website to help you craft sincere, thoughtful, and sometimes humorous apologies.

## Features

✨ **Multiple Tones**
- **Sincere & Heartfelt** - For genuine, deep apologies
- **Casual & Friendly** - For a relaxed approach
- **Formal & Professional** - For workplace or serious situations
- **Humorous & Light-hearted** - When a little humor helps

📝 **Easy to Use**
- Simple form to input the recipient's name, reason, and details
- Instant letter generation
- Customizable templates

💾 **Multiple Export Options**
- Copy to clipboard
- Download as text file
- Print directly

## Quick Start

1. Visit the website at https://cailycai.github.io/im-sorry-baby
2. Enter the recipient's name
3. Select your apology tone
4. Describe what you're apologizing for
5. Add any additional details (optional)
6. Click "Generate Letter"
7. Copy, download, or print your apology!

## How to Deploy

This site is deployed using GitHub Pages. It's automatically live at:
```
https://cailycai.github.io/im-sorry-baby
```

Any changes pushed to the `main` branch will automatically update the site.

## Technologies Used

- **HTML5** - Structure
- **CSS3** - Styling with gradients and responsive design
